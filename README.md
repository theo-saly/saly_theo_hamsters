# Symfony Hamster Game API

API backend pour un jeu de gestion de hamsters avec utilisateurs et gold.  
Deux rôles : `ROLE_USER` et `ROLE_ADMIN`.

---

## Prérequis

- PHP >= 8.1
- Composer
- Symfony CLI (optionnel mais recommandé)
- Base de données MySQL / MariaDB
- Node.js & npm (si vous utilisez webpack encore pour assets, pas obligatoire ici)
- Postman ou Insomnia pour tester l’API
